Shadowsocks软件介绍
SS，全称为 Shadowsocks（下文简称为SS），是一种基于 Socks5 代理方式的网络数据加密传输包的工具，可以使用其于科学上网 也就是翻墙 ，是目前最稳定，最常用的科学上网工具之一，全平台支持

SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

2017年7月19日，ShadowsocksR作者breakwa11在Telegram频道ShadowsocksR news里转发了深圳市启用SS协议检测并被大量用户转发，引发恐慌。7月24日，breakwa11发布了闭源的SS被动检测程序，引发争议。 2017年7月27日，breakwa11遭到自称 “ESU.TV” 的不明身份人士人身攻击，对方宣称如果不停止开发并阻止用户讨论此事件将发布更多包含个人隐私的资料，随后breakwa11表示遭到对方人肉搜索并公开个人资料的是无关人士，为了防止对方继续伤害无关人士，breakwa11将删除GitHub上的所有代码、解散相关交流群组，停止ShadowsocksR项目。但项目已被多人fork。并有人在其基础上继续发布新的版本，例如SSRR

Shadowrocket常见问题与使用指南
1. Shadowrocket规则文件
规则文件用于控制哪些流量走代理，哪些流量走直连。你可以在一些技术论坛或GitHub上找到相关的规则文件，常见的有：

GFWList：一个常用的规则列表，专门针对被屏蔽的网站。
自定义规则：你可以根据自己的需求，自定义规则文件。
2. PC端Shadowrocket
Shadowrocket并没有官方的PC版本。如果需要在PC上使用类似功能，可以考虑以下几种替代方案：

Shadowsocks：可以在Windows上安装Shadowsocks客户端，配置与Shadowrocket类似。
V2Ray：V2Ray也有PC端客户端，支持多种代理协议。
3. Shadowrocket IPA文件
IPA文件是iOS应用的安装包。如果需要安装Shadowrocket的IPA文件，可以通过以下方式：

从可信来源下载：请确保来源可靠，避免安全风险。
使用Cydia Impactor：将IPA文件通过Cydia Impactor安装到你的iOS设备上。
4. Shadowrocket PC版
正如前面提到的，Shadowrocket没有PC版。建议使用Shadowsocks或V2Ray等工具，这些工具在PC上能提供类似的科学上网功能。

5. Shadowrocket显示超时
如果Shadowrocket显示超时，可能有以下几种原因：

服务器问题：所选的代理服务器可能不稳定或已失效，尝试更换服务器。
网络连接：确保你的网络连接正常，可以尝试重启路由器或切换网络。
配置错误：检查代理设置，确认输入的服务器地址、端口等信息是否正确。
6. Shadowrocket规则下载
要下载Shadowrocket的规则文件，可以：

在线获取：访问一些技术论坛或GitHub，搜索相关的规则文件。
自定义规则：你也可以根据需要，自行编写规则文件，并导入Shadowrocket。
